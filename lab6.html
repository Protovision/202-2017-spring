Exercise 1

Create a slot machine game. The user will start off with 100 coins. The program should work according to this process:

  1. The user chooses to pay 1, 2, or 3 coins
  2. The slot machine spins. There are three reels that spin, each real can land on a number from 0 to 9.
  3. If all three reels land on the same number, the user wins coins equal to 300 multiplied by the number of coins they paid.

This process should repeat until the user chooses to quit or runs out of coins.

Name your source file "slots.cpp" and name your program "slots".

Exercise 2

Create a program that displays all the cards in a standard 52-card deck in a random order. The program should display each card on a seperate line in the form of "<RANK> of <SUIT>". For example, the first 5 lines of standard output might be:

8 of Spades King of Diamonds Ace of Hearts 3 of Hearts 4 of Diamonds ...

Name your source file "deck.cpp" and name your program "deck".
