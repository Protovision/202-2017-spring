Exercise 1

Create a program that reads a file (specified as a command-line argument) and displays it's characters in reverse order to the terminal. Name your source file "reverse.cpp" and name your program "reverse".

Exercise 2

Create a program that concatenates the lines of two files horizontally with a tab character joining each concatenated line. This program will work like the "paste" command in Bash. To get an idea of how paste works, suppose you have the following two files:

names.txt Mark Smith Bobby Brown Sue Miller Jenny Igotit numbers.txt 555-1234 555-9876 555-6743 867-5309

The command paste names.txt numbers.txt would produce the following standard output:

Mark Smith 555-1234 Bobby Brown 555-9876 Sue Miller 555-6743 Jenny Igotit 867-5309

This program should operate on two files specified on the command-line. Name your source file "paste.cpp" and name your program "paste".

Converting the argv array into a vector

It is not necessary, but you may find it useful to convert argv into a vector. Thus you may being your main function as follows:

int main(int argc, char *argv[]) { vector<string> args(argv, argv + argc);

After the args vector has been created, you can ignore argc and argv. Every element in args will be a C++ string copy of the C-strings existing in argv. args.size() will be equal to argc.
