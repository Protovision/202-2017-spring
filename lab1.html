Lab 1

Before doing this lab or any other lab assignment, please see the Cloud9 setup page.

Store all of your files for this lab in a folder named "lab1".

Exercise 1

Create a program that repeats it's input with all lines prepended with line numbers. Name your source file nl.cpp and name your program file nl.

Exercise 2

Create a program that counts the number of input words. Name your source file wc.cpp and name your program file wc.

Exercise 3

Create a program repeats it's input with all tab characters replaced with 4 space characters. Name your source file expand.cpp and your program expand.

Exercise 4

Create a program that repeats it's input with all blank lines removed. Name your source file nbl.cpp and your program nbl.
