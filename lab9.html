Exercise 1

Create a program that requires two parameters from the command-line. Each parameter will be a string representing a poker hand. Your program will display the hand that is the higher-ranking of the two hands. Name your program higher and your source file higher.cpp

Your program should be runnable from the command-line with a command such as the following:

./higher "3D 3S 3C KS 2S" "6H 6D 6S QC 4S"

For the above command, your program would display:

6H 6D 6S QC 4S

Because the 2nd hand ranks higher than the first

Poker hands can rank differently even though they have the same name (such as 2 hands that fall under "Two Pair"), please see this article on how poker hands are categorized and ranked between each other.
